# Security Clean Room

This is a fork of https://anonscm.debian.org/git/collab-maint/make-pgp-clean-room.git

## Usage

To use this, you need various utilities:

  apt-get install live-build debootstrap growisofs rsync sudo

To make a bootable CD/DVD:

   1. place a blank disk in the CD/DVD writer
   2. run the script scripts/make-pgpcleanroom
   3. when prompted by sudo, enter the root password

How it works
